# VeriQore

Blockchain-Based Academic Certificate Verification System

## Tech Stack

- React + Vite
- Node.js + Express
- MongoDB Atlas
- Solidity Smart Contracts
- Hardhat
- JWT Authentication

## Features

- Role-based authentication
- Master Admin Dashboard
- College Admin Management
- Student Certificate Management
- Recruiter Verification Portal
- Blockchain Certificate Verification
- Modern Space-Themed UI

## Setup

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend
npm install
npm run dev

### Blockchain

cd blockchain
npm install
npx hardhat compile
